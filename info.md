# ha-ulkovalot

Outdoor lights coordinator

Home Assistant integration installed via HACS from
[timmikk/ha-ulkovalot](https://github.com/timmikk/ha-ulkovalot) (public
GitHub mirror of the Forgejo primary at
[git.clo.dy.fi/timo/ha-ulkovalot](https://git.clo.dy.fi/timo/ha-ulkovalot)).

## Install

1. HACS → ⋮ → Custom repositories, add
   `https://github.com/timmikk/ha-ulkovalot`, category **Integration**.
2. HACS → Explore, find "Outdoor lights coordinator", Download.
3. Restart Home Assistant.
4. Settings → Devices & Services → Add Integration → Outdoor lights coordinator.

## Rollback

HACS → this integration → Redownload → pick prior version → restart HA.
